หัวข้อ: วงจร R-C High-pass พื้นฐาน

- วงจร high-pass แบบ R-C ใช้ capacitor อนุกรมกับ input และ resistor ต่อจาก output ลงกราวด์
- วัด output ที่แรงดันคร่อม resistor
- วงจรนี้ปล่อยให้สัญญาณความถี่สูงผ่านได้ดีกว่าสัญญาณความถี่ต่ำ
- เมื่อความถี่ต่ำมาก reactance ของ capacitor จะสูง ทำให้สัญญาณผ่านไปยัง output ได้น้อย
- ความถี่ตัดโดยประมาณคือ `fc = 1 / (2πRC)`
- ที่ความถี่ตัด ขนาดสัญญาณ output จะประมาณ `0.707` เท่าของ input หรือ `-3 dB`
- ค่าคงตัวเวลา คือ `τ = RC`
- วงจร high-pass ตอบสนองต่อการเปลี่ยนแปลงเร็วของสัญญาณได้ดี และมักใช้คัปปลิงสัญญาณ AC
