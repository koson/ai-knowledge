หัวข้อ: วงจร R-C Low-pass พื้นฐาน

- วงจร low-pass แบบ R-C ใช้ resistor อนุกรมกับ input และ capacitor ต่อจาก output ลงกราวด์
- วัด output ที่จุดต่อร่วมระหว่าง resistor และ capacitor
- วงจรนี้ปล่อยให้สัญญาณความถี่ต่ำผ่านได้ดีกว่าสัญญาณความถี่สูง
- เมื่อความถี่สูงขึ้น reactance ของ capacitor จะลดลง ทำให้สัญญาณถูกบายพาสลงกราวด์มากขึ้น
- ความถี่ตัดโดยประมาณคือ `fc = 1 / (2πRC)`
- ที่ความถี่ตัด ขนาดสัญญาณ output จะประมาณ `0.707` เท่าของ input หรือ `-3 dB`
- ค่าคงตัวเวลา คือ `τ = RC`
- การตอบสนองต่อ step ของแรงดันที่ capacitor มีรูป `Vc(t) = Vin(1 - e^(-t/RC))` สำหรับการชาร์จ
- หลังเวลาประมาณ `5τ` ระบบจะเข้าใกล้ค่าสภาวะคงตัว
